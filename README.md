# Recruit Client

A full-stack recruitment management application built as a personal project while expanding my React and modern web development skills.

## 🚀 Live Demo

**Demo:** https://renehu.net/recruit-client/#/login

> This application is deployed on my personal website and is available for demonstration.

## Features

- User authentication
- Candidate and recruitment management
- Responsive user interface
- RESTful API integration
- Real-time chat with Socket.IO
- CRUD operations
- Client-side routing
- Form validation

## Tech Stack

### Frontend

- React
- JavaScript (ES6+)
- React Router
- Axios
- HTML5
- CSS3

### Backend

- Node.js
- Express
- MongoDB
- Socket.IO
- RESTful APIs

## Related Project

Backend repository:
https://github.com/renehu/recruit-server

## Purpose

This project was developed as a personal project to expand my React and full-stack development skills. It demonstrates frontend-backend integration, RESTful APIs, MongoDB, and real-time communication using Socket.IO.

Most of my professional software development experience comes from enterprise banking systems, where the source code is protected by confidentiality agreements and cannot be shared publicly.
